# Plan — Template Premium Secteur 21 : Institution Publique & Gouvernementale

## Contexte
Création d'un template web premium React/Next.js pour les institutions publiques françaises (mairies, collectivités, services de l'État) conforme aux standards RGAA, DSFR, et accessibilité AA.

## Références
- `/mnt/agents/upload/sector-21-institution-publique.md` — Fiche secteur (structure, pages, fonctionnalités)
- `/mnt/agents/upload/21_institution_publique_gouvernementale.md` — Guide de conception détaillé (besoins métier, parcours, référentiels)

---

## Stages

### Stage 1 — Design System & Architecture
- Créer le design system (couleurs, typographie, spacing, composants)
- Définir l'architecture des pages et le routing
- Produire : `design.md` (design system complet)

### Stage 2 — Génération des Assets Visuels
- Générer les images nécessaires (hero institutionnel, photos de mairie, équipe, services publics)
- Produire : images dans `/mnt/agents/output/assets/`

### Stage 3 — Développement du Template (vibecoding-webapp-swarm)
- Construire le site React/Next.js avec toutes les pages publiques
- Pages : Accueil, Services, Démarches, Actualités, Politiques Publiques, Transparence, À Propos, Contact
- Implémenter le design system, animations, accessibilité, responsive
- Produire : application complète dans `/mnt/agents/output/`

### Stage 4 — Déploiement
- Déployer le site statique
- Produire : URL de production
